Where the Dimara and Perin focus on the human computer interaction services, giving tips on how the visualization/ artwork should attract the audience in the way the piece is setup, using the 7 tips and tricks that they give in their essay:

  1. Dialogue: this sees interaction as a cycle of communication acts, this is channelled through the input and the output from a computer perspective.
  2. Transmission: this sees interaction as a transmission of information between the computer and the user. This is to rate a error-free information per time unit that is transferred over a communication channel
  3. Control: sees interaction as minimization of error with respect to some user target. It is from a theory of continuous interaction via a system that consists of goals, inputs, outputs, feedbacks, feedforwards, and states. This is where the user is performing goal driven actions with the visualization system/ computer
  4. Tool use: sees interaction as a series of tools that let a user interact with a computer it can be a tool to manipulate or build this is based on the activity theory, and the instrumental interaction model. These tools are mandatory for the user and the object of interest.
  6. Optimal behaviour: sees interaction ad an adaptive behaviour under user capabilities, task, interface, and environment constrains. This is based on the rationality paradigm and adaptation theory. This how the humans and the systems can adapt to each other as the times moves, the behaviour that adapts and tests the systems.
  7. Embodiment: sees interaction as the act of being and participating in the world, this implied that the users should feel the technology, where the technology and the user are one thing that move and interact together, looking at the new tech such as VR where the user gets to interact with the visualization system.
  8. Experience: sees interaction as phenomenon shaped by the user experience, momentary reactions and recall this is based on the experience research, this is where the human experience is places as key factor in how interaction unfolds.

I implemented these 7 tips and tricks of how the audience interacts with the pieces, to make my spinning carrousel with the tools, it might spin with the code implemented in it but it work with giving the gist of the way that the website is setup and the way that it complements the website and the narrative that is being communicated on the website .
